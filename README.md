Social Book
1. Формулировка задачи
Цель проекта — создать веб-приложение Social Book, личный дневник с возможностью вести записи и оставлять комментарии к ним.
Задача заключается в том, чтобы пользователь мог добавлять, редактировать и удалять посты, а также управлять комментариями к каждому посту через удобный и отзывчивый интерфейс.
Приложение реализовано в виде SPA с централизованным хранилищем данных на основе Pinia, что обеспечивает согласованность состояния между компонентами без перезагрузки страницы.
2. На какую аудиторию рассчитываем
Основная аудитория Social Book — люди, которые хотят вести личный цифровой дневник с возможностью структурировать мысли и оставлять заметки в виде комментариев к записям.
Пользователь ожидает простой и чистый интерфейс, где можно быстро создать запись, отредактировать её через модальное окно и при необходимости удалить. Дополнительно к каждой записи можно развернуть список комментариев, добавить новый или убрать ненужный.
3. Что умеет приложение
Для каждого поста реализован полный набор действий:

Добавление поста через форму с полем ввода и кнопкой
Удаление поста по нажатию на кнопку
Редактирование поста через модальное окно с textarea, где можно изменить текст и сохранить или закрыть без изменений
Отображение и скрытие комментариев по нажатию на кнопку
Добавление и удаление комментариев внутри каждого поста

4. Как устроено приложение
Проект реализован на Vue 3 с использованием Composition API и Pinia в качестве хранилища.
Структура проекта:

stores/posts.js — централизованное хранилище Pinia со всеми постами и методами управления ими
components/AppButton.vue — кастомная кнопка с поддержкой вариантов (primary, danger, secondary и др.)
components/AppInput.vue — кастомный инпут с единым стилем и поддержкой v-model
components/PostItem.vue — карточка поста с логикой показа комментариев через локальный ref (isShow)
components/PostModal.vue — модальное окно редактирования поста, закрытие без сохранения не затрагивает данные в хранилище
App.vue — корневой компонент с формой добавления поста и списком всех записей

5. Как запустить
Установить зависимости и запустить dev-сервер:
npm install
npm run dev
Приложение откроется по адресу http://localhost:5173
